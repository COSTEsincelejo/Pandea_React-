# PANDEA es una plataforma de gestión de ventas orientada a microempresas, desarrollada con React y Vite.

El proyecto busca optimizar la administración de clientes, pedidos y ventas mediante herramientas digitales que facilitan el control de la información comercial y la gestión de pagos en línea.

# Tecnologías utilizadas
React
Vite
JavaScript
CSS3
Firebase
Git y GitHub
Wompi (pasarela de pagos)

# Funcionalidades
Gestión de clientes
Registro y seguimiento de pedidos
Administración de ventas
Autenticación de usuarios
Integración con Firebase
Procesamiento de pagos mediante Wompi
Interfaz web responsive
